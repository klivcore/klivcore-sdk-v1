# Minimal authenticated Realm

This template is generated by the Realm SDK owner. Copy the complete directory next to a clean `klivcore-sdk-v1` checkout, then follow the root SDK `README.md`.

Required runtime configuration:

- `REALM_PUBLIC_ORIGIN`: exact browser-visible HTTPS origin;
- `PORT`: private loopback port, default `47002`;
- `REALM_AUTH_DATABASE`: optional Realm-owned SQLite path;
- `REALM_REGISTRATION_FILE`: optional new mode-`0600` file for one-time first-user registration.

The template deliberately fails closed when the exact public origin is absent. For a Quick Tunnel, start `cloudflared` first, obtain its generated origin, and then start this Realm with that exact value.

```bash
bun install
bun test
bun run build
REALM_PUBLIC_ORIGIN=https://generated-name.trycloudflare.com bun run dev
```

Never log or commit the registration URL, authentication database, cookies, passkeys, or session material.
